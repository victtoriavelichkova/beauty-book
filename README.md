# BeautyBook - Hair Salon System

## Tech Stack
- Java 17
- Spring Boot 3.4
- Spring MVC
- Spring Security
- Spring Data JPA
- Thymeleaf
- MySQL

## Features
- Client management (CRUD)
- Hairdresser management (CRUD)
- Services management (CRUD)
- Appointment booking system
- Role-based access (ADMIN / USER)

## Domain Entities

- User
- Client
- Hairdresser
- SalonService
- Appointment

## How to run
1. Create MySQL database: beauty_book_db
2. Run application
3. Open: http://localhost:8080
4. Login with:
    -  username: admin
    -  password: admin
